# NODEJS + MONGODB (MONGOOSE)

* Clone
```
$ git clone https://github.com/Elidjango/nodejs-mongodb.git
$ cd nodejs-mongodb
```

* Install Package
```
$ npm install --save --force
```

after installing the packages we proceed to configure dotenv in the .env file

* Run API
```
$ npm run start:dev
```
